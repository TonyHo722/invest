Stock Split Detection & Price Adjustment

Implementation Architecture in yf_proxy.py

📊 Introduction to the Split Problem

A stock split changes the number of outstanding shares and the stock price proportionally, while leaving the company's total market value unchanged. For example, in a 2-for-1 forward split, the stock price drops by 50% while the share count doubles.

If historical price databases are not adjusted backwards to reflect this split, historical technical indicators (like the 200-Day Moving Average) will show artificial price jumps or drops, rendering technical screening results incorrect.

1. Official Split Adjustment

Retrieves split occurrences using the official Yahoo Finance Stock Splits column. If a split date is matched, historical prices prior to that date are divided (or multiplied) by the split factor.

2. Heuristic Auto-Detection Fallback

Under international markets (like TWSE or JPX), Yahoo Finance often fails to record stock splits officially. The proxy uses a mathematical sliding-window algorithm to detect integer-ratio price jumps and adjust historical data retroactively.

🛠️ Technical Implementation Flow

The split adjustment sequence operates as follows:

  1. Chronological Sort: Orders the historical data frame in ascending order (past to present).
  2. Ratio Scan: Iterates through the daily close prices, comparing today's price ($P_{curr}$) to yesterday's price ($P_{prev}$).
  3. Tolerance Filter: Checks if the ratio matches common split factors (e.g. 2, 3, 4, 5, 10) within a 15% margin of tolerance.
  4. Conflict Verification: Evaluates an 11-day window surrounding the day of the price jump to ensure no official split was already recorded.
  5. Historical Correction: Multiplies or divides all historical prices ('Open', 'High', 'Low', 'Close') prior to that date to smooth out the transition.

💻 Code Snippet: Detection & Adjustment Logic

The core python functions from scripts/yf_proxy.py are structured below:

def auto_detect_splits_single(df):
    if df.empty or 'Close' not in df.columns:
        return df
        
    df = df.copy()
    if isinstance(df.index, pd.DatetimeIndex):
        df = df.sort_index()
        
    # Common forward/reverse split factors
    factors = [2, 3, 4, 5, 6, 7, 8, 10, 20, 50, 100]
    
    for i in range(1, len(df)):
        p_prev = df.iloc[i-1]['Close']
        p_curr = df.iloc[i]['Close']
        
        if pd.isna(p_prev) or pd.isna(p_curr) or p_prev <= 0 or p_curr <= 0:
            continue
            
        ratio = p_prev / p_curr
        
        # 1. Check for unrecorded FORWARD splits
        for R in factors:
            if abs(ratio - R) < 0.15 * R:
                start_win = max(0, i - 5)
                end_win = min(len(df), i + 6)
                has_official_split = False
                
                if 'Stock Splits' in df.columns:
                    window_splits = df.iloc[start_win:end_win]['Stock Splits'if ((window_splits.notna()) & (window_splits > 0) & (window_splits != 1.0)).any():
                        has_official_split = True
                
                if not has_official_split:
                    drop_date = df.index[i]
                    print(f"Auto-detected split of {R}:1 on {drop_date.date()}...")
                    for metric in ['Open', 'High', 'Low', 'Close']:
                        if metric in df.columns:
                            df.iloc[:i, df.columns.get_loc(metric)] /= R
                    break
                    
        # 2. Check for unrecorded REVERSE splits
        rev_ratio = p_curr / p_prev
        for R in factors:
            if abs(rev_ratio - R) < 0.15 * R:
                start_win = max(0, i - 5)
                end_win = min(len(df), i + 6)
                has_official_split = False
                
                if 'Stock Splits' in df.columns:
                    window_splits = df.iloc[start_win:end_win]['Stock Splits'if ((window_splits.notna()) & (window_splits > 0) & (window_splits != 1.0)).any():
                        has_official_split = True
                        
                if not has_official_split:
                    jump_date = df.index[i]
                    print(f"Auto-detected reverse split of 1:{R} on {jump_date.date()}...")
                    for metric in ['Open', 'High', 'Low', 'Close']:
                        if metric in df.columns:
                            df.iloc[:i, df.columns.get_loc(metric)] *= R
                    break
    return df

💡 Benefits of This Approach

  • Robust Handshaking: Blends both official splits and raw price movement checks to avoid double-adjustments.
  • Clean Moving Averages: Guarantees that indicator computations (like 200-DMA) are never corrupted by artificial splits.
  • Multidimensional Safe: Works seamlessly on both single-ticker data frames and MultiIndex data structures (batch mode).
